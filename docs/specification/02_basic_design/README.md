# 基本設計

機能設計と画面設計をまとめるディレクトリです。

- 01_function_design/: 機能設計一式。
  - 01_function_list/: 機能一覧（AUTH/USR/CAT/TAG/THM/NOTE）と関連画像。
  - 02_business_flow/: 業務フロー整理と関連画像。
- 02_screen_design/: 画面設計一式（画面一覧・フロー・サンプル・画像）。
