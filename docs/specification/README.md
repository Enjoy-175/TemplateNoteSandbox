# 仕様ドキュメント概要

この配下は仕様関連の資料をまとめたディレクトリです。目的別に3階層へ分割しています。

- 01_requirement_definition: 要件定義。背景・目的・前提などの整理。
- 02_basic_design: 基本設計。機能設計と画面設計の整理。
- 03_detail_design: 詳細設計。API・処理・DBの具体設計。

```
docs/specification
├── 01_requirement_definition/
├── 02_basic_design/
├── 03_detail_design/
└── README.md
```
